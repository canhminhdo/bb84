# Formal Specification of the BB84 Protocol in Maude and CafeOBJ
This repository contains the formal specification of the BB84 protocol in Maude and CafeOBJ for model checking and theorem proving of desired properties of the protocol, respectively.

### Introduction
- For model checking, please refer to `maude/bb84.maude`
- For theorem proving, please refer to `cafeobj/bb84.cafe` and the formal proofs in `cafeobj/proof-scores` folder.

### Publications
- C.M. Do and K. Ogata, "Formal Specification and Model Checking of the BB84 Protocol in Maude", SEKE 2025. (DOI: [10.18293/SEKE2025-020](https://doi.org/10.18293/SEKE2025-020))
